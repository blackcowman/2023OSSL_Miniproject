도서관 사서와 이용자 시스템
========
![book-g147f3075e_1280](https://github.com/blackcowman/2023OSSL_Miniproject/assets/90015973/1bc39265-aa88-486c-81c2-567624438ecd)
### 목적
+ 도서관 및 도서 대출 관리
+ 사서와 이용자의 입장에서 도서관 사용

### 기능 (개발언어: C)
1. 사서 및 이용자 로그인
2. 메뉴 확인(사서 전용, 이용자 전용)
3. 사서: 도서관리(추가, 삭제, 수정), 대출된 책 목록 확인
4. 이용자: 도서 대출, 예약 및 반납
5. 도서목록 조회 및 검색

### 도서정보
1. 제목
2. 저자
3. 출판사
4. 도서코드
5. 대출여부
6. 예약여부
